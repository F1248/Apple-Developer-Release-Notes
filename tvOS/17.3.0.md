# tvOS 17.3 Release Notes

Update your apps to use new features, and test your apps against API changes.

## [Overview](https://developer.apple.com/documentation/tvos-release-notes/tvos-17_3-release-notes#Overview)

The tvOS 17.2 SDK provides support to develop tvOS apps for Apple TV devices running tvOS 17.3. The SDK comes bundled with Xcode 15.1, available from the Mac App Store. For information on the compatibility requirements for Xcode 15.1, see [Xcode 15.1 Release Notes](https://developer.apple.com/documentation/Xcode-Release-Notes/xcode-15_1-release-notes).
